Instruções de Instalação do Vale Cashback para Android

Importante: Para instalar o aplicativo Vale Cashback no Android, você precisará permitir a instalação de aplicativos de fontes desconhecidas. Isto é seguro para o nosso aplicativo.

Passo 1: Baixar o APK

Clique no botão "Baixar APK" na página de downloads. O arquivo vale-cashback-android.apk será baixado para o seu dispositivo.

Passo 2: Abrir o arquivo APK

Depois que o download for concluído, abra o arquivo APK. Você pode encontrá-lo na pasta de downloads do seu dispositivo ou tocar na notificação de download completo.

Passo 3: Permitir instalação de fontes desconhecidas

Se esta for a primeira vez que você instala um APK diretamente, seu dispositivo solicitará permissão para instalar aplicativos de fontes desconhecidas.

Toque em "Configurações" e ative a opção "Permitir desta fonte" ou similar, dependendo da versão do Android.

Passo 4: Concluir a instalação

Depois de conceder a permissão, toque em "Instalar" para iniciar a instalação do aplicativo.

Quando a instalação for concluída, toque em "Abrir" para iniciar o Vale Cashback.

Dica: Se você encontrar algum problema durante a instalação, tente baixar o arquivo novamente ou entre em contato com nosso suporte técnico através do e-mail suporte@valecashback.com
